# House Price Prediction using Linear Regression

## Project Overview
This project predicts house prices using a Linear Regression model.
The dataset contains information such as number of bedrooms, bathrooms,
living area, lot size, floors, condition, and year built.

The goal of this project is to understand the complete machine learning
workflow including data analysis, preprocessing, model training, and evaluation.

---

## Dataset
- Total records: 4600
- Target variable: price

---

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

---

## Steps Performed
1. Data loading and exploration
2. Data cleaning and removal of unnecessary columns
3. Correlation analysis using heatmap
4. Outlier detection using Z-score
5. Train-test split
6. Linear Regression model training
7. Model evaluation
